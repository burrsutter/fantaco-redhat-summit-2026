How themes work

  • Universal — no rows in product_pod_theme (empty podThemes). These SKUs apply to every workspace theme.
  • Explicit tags — one or more theme names. The SKU is marketed for those pods.
  • Applicable to theme — the API returns universal SKUs plus any SKU that lists that theme.
